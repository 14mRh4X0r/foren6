h2. Debian packages

Download the appropriate package:
* "*foren6_1.0-1_i386.deb*":http://sourceforge.net/projects/foren6/files/foren6-1.0-1/foren6_1.0-1_i386.deb/download
* "*foren6_1.0-1_amd64.deb*":http://sourceforge.net/projects/foren6/files/foren6-1.0-1/foren6_1.0-1_amd64.deb/download

Install the package:
<pre>sudo dpkg -i foren6_1.0-1_arch.deb</pre>

If this is your first installation, it is likely you are missing dependencies. Run the following to fix and finish the installation.
<pre>sudo apt-get install -f</pre>

h2. Linux/MacOS X, from source

Install dependencies<pre>sudo apt-get install libqt4-dev qt4-qmake cmake make libexpat1-dev tshark
libpcap0.8-dev libc6-dev g++ gcc</pre>

Obtain sources and install<pre>git clone https://github.com/cetic/foren6.git
cd foren6
make
sudo make install</pre>
